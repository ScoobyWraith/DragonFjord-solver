## Решение головоломки Dragon Fjord calendar (бесконечный календарь)

![Головоломка](https://dracos.co.uk/wrote/a-puzzle-a-day-january/puzzle-small.jpeg)

Решение написано на TypeScript. 

Для UI/UX используется **React**.

Сборка проекта из исходников осуществляется с помощью **vite**.
Команда для сборки: `npm run build`.

Посмотреть пример: https://scoobywraith.github.io/DragonFjord-solver/
